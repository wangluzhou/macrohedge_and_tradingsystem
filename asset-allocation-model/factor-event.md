# 因子事件

## 因子事件备忘条目

1. 分成四种因子事件：
   * 单边正向事件
   * 单边负向事件
   * 双边正向事件
   * 双边负向事件
2. 对于单边事件，事件信号必须为正值
3. 单边信号的准确率达到60%，即为合格，达到70%为优秀
4. 双边信号因子目前可以用超额夏普比率进行评判
